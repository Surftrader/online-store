# Завдання 1
Створіть Django-застосунок, інтернет-магазин.  

**Вимоги:**
- Створити сторінку, де виводитимуться всі товари.
- Створити сторінку, де виводитиметься інформація про один товар.
- Додати можливість авторизованому користувачеві залишати відгук/коментар про товар.
- Сторінка реєстрація/авторизація.
- Фільтрування за категоріями товарів.
- Додати можливість пошуку товару на сторінці.
- Створити сторінку для оформлення замовлення для авторизованих користувачів (без платіжної системи).

**Додаткові вимоги:**
- Можливість підписки на новини (celery).
- Команда для завантаження із зовнішньої БД (команда для manage.py).
- Переглянуті товари.  

Стилізація – на Ваш розсуд.  

**Важливі моменти:**
- Блокування переходу за меню edit/delete/create, якщо користувач не авторизований.
- Використання SQL.
